# CNN Image Classification using PyTorch

## Project Overview
This project implements a Convolutional Neural Network (CNN) using PyTorch for image classification on the CIFAR-10 dataset.

The model is trained on colored images from 10 different classes and achieves approximately 74.71% accuracy on the test dataset.

---

## Features
- Convolutional Neural Network (CNN)
- CIFAR-10 Dataset
- PyTorch Deep Learning Framework
- Adam Optimizer
- CrossEntropyLoss
- Model Evaluation and Accuracy Calculation
